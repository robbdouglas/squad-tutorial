Aufgabe: Verwaltung von Büchern in einer Bibliothek mit MongoDB

Teil 1: Datenmanipulation

Erstellen Sie eine neue Datenbank namens "library".
Erstellen Sie zwei Sammlungen: "books" und "authors".
Fügen Sie einige Beispieldokumente in jede Sammlung ein, die relevante Informationen enthalten, z. B. Buchtitel, Autorinformationen.

Teil 2: Einfache Abfragen

Suchen Sie nach Büchern eines bestimmten Genres.
Finden Sie heraus, welche Bücher von einem bestimmten Autor geschrieben wurden.
Ermitteln Sie alle Bücher, die nach einem bestimmten Jahr veröffentlicht wurden.

Teil 3: Aggregation und Analyse

Ermitteln Sie die Anzahl der Bücher pro Autor.
Finden Sie heraus, welches Genre die meisten Bücher hat.
Berechnen Sie den durchschnittlichen Bewertungswert aller Bücher.

Teil 4: Datenmodellierung und Beziehungen

Verknüpfen Sie die "books" und "authors" Sammlungen über die Autoren-ID in den Buchdokumenten.
Fügen Sie einige zusätzliche Felder zu den Dokumenten in der "books" -Sammlung hinzu, z. B. "veröffentlichtes Jahr".

Teil 5: Komplexere Abfragen und Aggregation

Ermitteln Sie die Anzahl der Bücher pro Jahr.
Finden Sie heraus, welches Jahr die meisten Bücher veröffentlicht wurden.
Listen Sie alle Bücher auf, die zwischen zwei bestimmten Jahren veröffentlicht wurden und eine Bewertung von mindestens 4,0 haben.
Erstellen Sie geeignete Indizes für die Felder, die in Ihren Abfragen verwendet werden.
Überprüfen Sie die Leistung der Abfragen vor und nach der Indizierung.
